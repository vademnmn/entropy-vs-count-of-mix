# entropy-vs-count-of-mix
a numerical model of reversible heat exchange demonstrating entropy convergence behavior as the number of thermal partitions increases.

# Reversible Heat Exchange and Entropy Simulation

This project investigates the dependence of entropy change on the number of partitions in a reversible heat exchange model.

The system consists of two bodies with different initial temperatures divided into multiple smaller parts. Heat exchange is performed sequentially between these partitions, allowing the process to approach thermodynamic reversibility as the number of partitions increases.
